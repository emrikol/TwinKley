# TwinK[l]ey - GitHub Pages

This branch hosts the public-facing content for TwinKley:

- **index.html** - Landing page at https://emrikol.github.io/TwinKley/
- **appcast.xml** - Sparkle update feed (auto-generated by GitHub Actions on each release)

## URLs

- Landing page: https://emrikol.github.io/TwinKley/
- Appcast feed: https://emrikol.github.io/TwinKley/appcast.xml

## Notes

- The appcast.xml is managed automatically by the release workflow - do not edit manually
- The index.html can be updated manually as needed
